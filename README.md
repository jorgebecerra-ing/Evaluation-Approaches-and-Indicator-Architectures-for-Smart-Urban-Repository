# Smart Urban Mobility Evaluation

## Overview
Rapid urbanization has intensified congestion, environmental pressures, and transport inequities. This has increased interest in **smart urban mobility** as an approach that combines:
- Digital technologies  
- Sustainable transport strategies  
- Data-informed decision-making  

to address these challenges.

## Problem
Evaluation of smart urban mobility remains **fragmented** due to:
- Lack of harmonized assessment frameworks  
- Diversity of methodologies across smart city contexts  

## Study
This project presents a **systematic literature review** of evaluation approaches and indicator architectures for smart urban mobility in smart city contexts.

- **Methodology:** PRISMA-guided screening process  
- **Records retrieved:** 412  
- **Eligible studies:** 33  

### Identified Methodological Groups
1. Quantitative approaches  
2. Multi-criteria decision-making methods  
3. Qualitative or participatory frameworks  

### Indicators
- **Total:** 273 indicators  
- **Organized into:** 8 factor categories  
- Confirms the **multidimensional nature** of smart mobility assessment  
- Reveals limited consistency in indicator selection and application  

## Findings
Current evaluation practices are increasingly linked to:
- Project prioritization  
- Planning  
- Decision support  

However, effectiveness is constrained by:
- Data inconsistencies  
- Governance fragmentation  
- Insufficient user inclusion  

## Conclusion
There is a clear need for **assessment frameworks** that are:
- Comparable enough to enable **cross-city learning**  
- Flexible enough to reflect **local contexts** and **institutional realities**
